# Assets

- `images/logo-oficial.png`: logo enviada pelo cliente.
- `images/brand-cover.webp` e `images/og-cover.jpg`: imagens de apresentação e compartilhamento.
- `images/product-placeholder.svg`: fallback para produtos sem foto.
- `images/barbershop-placeholder.svg`: fallback para foto da barbearia.
- `icons/`: ícones locais da interface.
- `fonts/`: orientações de tipografia.

As fotos reais podem ser cadastradas pelo painel sem alterar a estrutura do front.
